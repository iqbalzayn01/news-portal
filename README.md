# News Portal

## Instalasi

1. Klon repositori:

```
git clone https://github.com/iqbalzayn01/news-portal nama_folder
```

2. Instal dependensi:

```
cd news-portal
npm install
```

3. Jalankan project:

```
npm run dev
# buka browser dan kunjungi http://localhost:3000
```

### Alat yang Digunakan

- [Vue](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [News API](https://newsapi.org/)
- [TailwindCSS](https://tailwindcss.com/)
